Lista de Tags em HTML5

Doctype html
Indica ao navegador qual versão do HMTL está sendo utilizada
HTML
Tag que contém todos os demais elementos do arquivo HTML
Head
Tag que delimita o cabeçalho do documento
Title
Tag que define o título da página, isto é, o nome que aparecerá na aba do navegador.
Meta
Essa tag permite inserir metadados ao seu documento, no caso acima, a informação charset=”UTF-8″, que garante a compatibilidade do código com os caracteres de padrão latino americano
body
Tag que delimita o corpo do documento. Todos os elementos visíveis do site são escritos dentro desta tag.
h1..h6
Insere títulos de nível um até nível seis.
p
Insere parágrafo.
hr
Insere linha horizontal.
br
Insere quebra de linha.
img
Insere imagem.
b
Coloca o texto em negrito.
strong
Coloca o texto em negrito.
i
Coloca o texto em itálico.
em
Coloca o texto em itálico.
mark
Marca um texto com uma cor de fundo...
small
Coloca um texto em letras miúdas.
del
Marca um texto como tachado.
u
Sublinha um texto.
ins
Sublinha um texto.
sup
Coloca um texto em sobrescrito.
sub
Coloca um texto em subescrito.
pre
Esta tag escreve um texto pré-formatado, isto é, escreve o texto da mesma forma que o texto original foi disposto no arquivo html. Espaços em branco são mantidos no texto da mesma forma em que este foi digitado.
code
É usada para colocar códigos.
q
É usada para inserir citações simples.
blockquote
É usada para inserir citações em bloco.
abbr
Indica que uma palavra é uma abreviação. Além disso, através do parâmetro title, coloca a definição de uma abreviação.
ol
Cria uma lista ordenada.
ul
Cria uma lista não ordenada.
li
Adiciona o elemento de uma lista.
dl
Cria uma lista de definição.
dt
É usada para indicar o termo que será definido pela lista de definição.
dd
Representa a descrição do termo colocado na tag <dd>.
